# Smart Parking Finder UI 

## Overview
Designed a modern UI/UX concept for a smart parking application that helps users quickly find and reserve nearby parking spots.

---

## Problem
Finding parking in busy areas is time-consuming and inefficient.

---

## Solution
A clean interface that shows:
- Available parking spots
- Pricing and availability
- Quick reservation option

---

## Features
- Search location input
- Map view placeholder
- List of available parking spots
- Reserve button interaction
